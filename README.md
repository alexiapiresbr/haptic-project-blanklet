# haptic-project-blanklet
Haptic technology to distract small children during small medical procedures

The goal of this project is to create a visual distraction on a touchscreen by developing a
game that is connected to haptic actuators placed on the child’s belly. The aim for the end
user (the child) is to stay engaged with the game while also physically feeling the gameplay
through haptic feedback on their belly.
To achieve this, we will develop several games that can be controlled by tilting the screen.
An IMU sensor will be attached to the screen to detect its orientation. A calibration system
will be implemented to accurately collect and interpret the input data generated by the tilting
motion.

The microcontroller connected to the screen will process the game data and the IMU input,
and then send this information to a second microcontroller located in the blanket. This
microcontroller will control servos and vibration actuators, which will create the haptic
sensations that correspond to events in the game, allowing the child to feel the gameplay on
their belly.
Additionally, a passive buzzer will be integrated into the Arduino Micro. The goal is to
implement this buzzer so the child receives a sensory audio response from the game,
synchronized with the events on screen. This is achieved using PWM to generate different
tones for alerts or successes, increasing the child's engagement with the device.
